DATASET / TRSET 가이드

※ DATASET, TRSET 사용시 유의사항

1. DATASET, TRSET 생성할때 유일한 ID값을 설정해야 한다.
2. DATASET, TRSET의 Function을 사용할때에는 Function명 대소문자를 정확하게 입력해야 한다.

  • DataSet Script
  • DataSet / TrSet
  • DATASET Function

DataSet Script 선언

※ Gauce Script 선언 및 include 선언 방법을 JavaScript Object 생성으로 변경
- bl에서 sql이 generateSql 기능으로 조회될 경우 dataSet의 true 적용

AS-IS 
<script language="javascript" src="../aviscmn/js/grid.js"></script>
<script language="javascript" src="../aviscmn/js/dataset.js"></script>
<script language="javascript" src="../aviscmn/js/transset.js"></script>

<%@ include file="../aviscmn/inc/condSet.jsp"%>
<%@ include file="../aviscmn/inc/dataSet01.jsp"%>
<%@ include file="../aviscmn/inc/dataSet02.jsp"%>
<%@ include file="../aviscmn/inc/trSet01.jsp"%>
<%@ include file="../aviscmn/inc/bindCtrl01.jsp"%>
TO-BE 
<script type="text/javascript">
    var condSet   = new DATASET("condSet");
    var dataSet01 = new DATASET("dataSet01");
    var dataSet02 = new DATASET("dataSet02");
    var trSet01   = new TRSET("trSet01");
</script>
TO-BE (generateSql) 
<script type="text/javascript">
    var condSet   = new DATASET("condSet");
    var dataSet01 = new DATASET("dataSet01", true); // dataSet01이 generateSql 기능으로 조회 될 경우 true 적용
    var dataSet02 = new DATASET("dataSet02");
    var trSet01   = new TRSET("trSet01");
</script>

function initWindow() {

function initWindow() {

 Gauce DataSet → Script DataSet Function

※ Gauce DataSet 에서 사용하던 Function 명은 동일하나 사용법은 변경
-  Setter Function (ColumnString, ColumnValue, NameString, NameValue 등..)
    DATASET Object에 값을 삽입할 경우 기본방식과 다르게 특정함수에 파라미터값을 넘기도록 변경

AS-IS 
condSet.NameString(1, "DEPTCODE")   = obj.DEPTCODE.value;
dataSet.NameValue(1, "DEPTCODE")    = obj.DEPTCODE.value;
dataSet.ColumnString(1, "DEPTCODE") = obj.DEPTCODE.value;
dataSet.ColumnValue(1, "DEPTCODE")  = obj.DEPTCODE.value;

/* IBSheet 매핑된 DataSet */
// 값 설정
dataSet.NameString(1, "DEPTCODE")   = obj.DEPTCODE.value;
dataSet.NameValue(1, "DEPTCODE")    = obj.DEPTCODE.value;

// 값 가져오기
dataSet.NameString(1, "DEPTCODE");
dataSet.NameValue(1, "DEPTCODE");
TO-BE 
condSet.NameString(1, "DEPTCODE", obj.DEPTCODE.value);
dataSet.NameValue(1, "DEPTCODE", obj.DEPTCODE.value);
dataSet.ColumnString(1, "DEPTCODE", obj.DEPTCODE.value);
dataSet.ColumnValue(1, "DEPTCODE", obj.DEPTCODE.value);

/* IBSheet 매핑된 DataSet */
// 값 설정
sheet.SetCellValue(1, "DEPTCODE", obj.DEPTCODE.value);
// 값 가져오기
sheet.GetCellValue(1, "DEPTCODE");


※ 유의사항

- 대소문자를 정확하게 입력

AS-IS 
dataSet.namevalue(1, "DeptCode", obj.DEPTCODE.value);
dataSet.namestring(1, "DeptCode", obj.DEPTCODE.value);
dataSet.columnstring(1, "DeptCode", obj.DEPTCODE.value);
dataSet.columnvalue(1, "DeptCode", obj.DEPTCODE.value);
TO-BE 
dataSet.NameValue(1, "DEPTCODE", obj.DEPTCODE.value);
dataSet.NameString(1, "DEPTCODE", obj.DEPTCODE.value);
dataSet.ColumnString(1, "DEPTCODE", obj.DEPTCODE.value);
dataSet.ColumnValue(1, "DEPTCODE", obj.DEPTCODE.value);
