commonIBMDITab.js

IBMDITab 설정시 사용되는 자바스크립트.
화면 크기를 변경되 Tab 크기가 변경 될 경우 OnResize이벤트가 과도하게 발생하는 것을 0.2초 간격으로 발생하도록 조절해 주는 기능이 내장되어 있음.

initMainTab(menuTab, isCloseBtn, tabResizeCallFunction)

IBMDITab 메인화면 탭 설정 Function (메인화면 전용)
tabResizeCallFunction 파라미터값 미 설정시 기본 이벤트로 설정

menuTab
IBMDITab Object
isCloseBtn
탭 닫기 버튼(boolean)
tabResizeCallFunction
탭 화면사이즈 조정시 호출할 Function

initComTab(menuTab, tabTitleArry, isCloseBtn, tabResizeCallFunction)

IBMDITab 초기화 탭 설정 Function

menuTab
IBMDITab Object
tabTitleArry
Tab Title Array
isCloseBtn
탭 닫기 버튼(boolean)
tabResizeCallFunction
탭 화면사이즈 조정시 호출할 Function
