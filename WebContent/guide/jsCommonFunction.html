AJ렌터카 개발 가이드

commonFunction.js

공통으로 사용되는 Map Object, Ajax 기능, Data Mapping 기능을 담고 있는 자바스크립트.
화면 크기를 변경할 경우 OnResize이벤트가 과도하게 발생하는 것을 0.2초 간격으로 발생하도록 조절해 주는 기능이 내장되어 있음.

Map

Java에서 사용하는 Map Class를 동일한 기능을 가진 JavaScript용 Map Object (java 사용법 동일)

  • put
  • get
  • containsKey
  • containsValue
  • clear
  • remove
  • keys
  • values
  • size

ComEJBSync(ejbName, methodName, paramData)

EJB를 호출하는 Ajax Sync Function DATASET/TRSET을 사용하지 않음.

ReturnData
JSON 형태의 Object
ejbName
EJB Class명
methodName
EJB Method명
paramData
파라미터를 넘길 JSON 형태의 Object

ComEJBAsync(ejbName, methodName, paramData, callBackFun)

EJB를 호출하는 Ajax Async Function DATASET/TRSET을 사용하지 않음.

ReturnData
JSON 형태의 Object
ejbName
EJB Class명
methodName
EJB Method명
paramData
파라미터를 넘길 JSON 형태의 Object
callBackFun
데이터처리 이벤트 완료 후 호출되는 Function

ComSetBindJsonData(jsonData, defaultValue)

JSON 데이타를 받아서 해당되는 KEY로 객체의 ID를 찾아서 VALUE를 셋팅해주는 Function

ComEJBSync, ComEJBAsync 사용하여 넘어온 결과값을 사용시 사용
jsonData
JSON 형태의 Array
defaultValue
JSON 데이터의 값이 Null 또는 undefined 값 일 경우 기본값으로 설정할 값

ComSetObjValue(htmlId, value)

Html Id에 해당하는 객체를 찾아서 value를 셋팅해주는 Function

htmlId
HTML id명
value
설정값

defaultString(obj)

Object 값이 NULL 또는 UNDEFINED이면 ""를 반환하고 아니면 Object에 설정되어 있는 값을 반환

returnData
"" 또는 Object 값
obj
데이터를 확인할 Object

defaultNumber(obj)

Object 값이 NULL 또는 UNDEFINED이면 0를 반환하고 아니면 Object 값을 int형으로 반환

returnData
0 또는 Object 값을 변환한 int형 값
obj
데이터를 확인할 Object
