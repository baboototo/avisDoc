AJ렌터카 개발 가이드

Eclipse Plug-In

  Eclipse Plug-In Setting
이클립스 네비게이션 또는 패키지 익스플로러에 작업파일(JSP)에만 AJRentacar 메뉴가 추가된다.
추가된 메뉴는 작업파일(JSP)을 읽어서 특정부분 소스를 추출하여 이클립스 뷰에 추출한 소스를 뿌린다.

1. AJ렌터카 개발 가이드 우측상단에 있는 다운로드 메뉴에서 Eclipse Plug-In.jar 파일을 다운로드 받은 후 Eclipse Plug-In.jar 파일을 C:\ADT_HOME\ide\eclipse_wtp_2.0\plugins 안에 복사한다.

2. 이클립스를 재시작 한다. 로그를 확인할 JSP파일을 선택 후 마우스 오른쪽 버튼을 클릭하여 AJRentacar 메뉴에 속해 있는 메뉴를 선택한다.

3. Compare Gauce/DataSet Column 메뉴는 AS-IS화면의 Grid와 DataSet 맵핑정보 및 소스에서 사용하는 DataSet 컬럼을 추출하여 AJRentacar 로그창에 로그를 뿌린다.

4. Compare IBSheet Column 메뉴는 TO-BE화면의 IBSheet 및 소스에서 사용하는 IBSheet 컬럼을 추출하여 AJRentacar 로그창에 로그를 뿌린다.

5. Check HTML ID 메뉴는 HTML 태그에서 ID 속성이 설정되어 있는 ID명을 추출하여 AJRentacar 로그창에 로그를 뿌린다.

6. Check HTML Hidden Input 메뉴는 HTML 태그에서 Hidden 속성으로 설정되어 있는 HTML 추출하여 AJRentacar 로그창에 로그를 뿌린다.

7. Check JS getElementById 메뉴는 getElementById 스크립트를 사용한 소스를 추출하여 AJRentacar 로그창에 로그를 뿌린다.

8. 이클립스 Window > Show View > Other에서 AJRentacar 로그창을 열수도 있다. (위 3-7 메뉴 선택시 로그창 자동오픈됨)
