TRSET API

Gauce TRSET Object를 기반으로 커스터마이징한 JavaScript Object이다.

* TRSET 사용시 유의사항
TRSET 생성시 유일한 ID값을 설정해야 한다.
TRSET Function 또는 속성값 사용시 대소문자를 정확하게 입력해야 한다

TRSET(trSetId)

new TRSET("trSetId") 생성시 dataSetId 값을 필수로 설정해야 한다.

dataSetId
IBSheet와 DATASET 맴핑시 사용하는 KEY
isGenerateDataType
기본값은 false이며, EJB BL에서 generateSql 메소드 사용시 true로 설정
(최초 조회한 DATASET 헤더값으로 변경하여 데이터를 서버로 전송하기 떄문에 true 설정해야만 기존과 동일하게 사용할 수 있다.)

DATASET 속성 변수값

id
DATASET ID
CountRow
DATASET Row 총 개수
CountColumn
DATASET Column 총 개수

Reset()

DATASET 초기화

SetDataHeader(headerStr)

데이터 CRUD 이벤트 발생시 서버로 전송할 데이터 헤더(컬럼)명 설정

headerStr
데이터 헤더(컬럼)명 ("CUSTNAME:STRING(50),CUSTCODE:STRING(10)")

ClearAll()

DATASET 데이터 초기화 및 헤더 데이터 초기화

ClearData()

DATASET 데이터 초기화 (헤더 데이터 제외)

SetDataSet(dataSetArrObj)

DATASET에 데이터와 헤더 정보를 설정 (데이터 조회한 JSON 데이터를 이용해 DATASET에 값을 담을 경우에만 사용)

dataSetArrObj
JSON 형태의 Array Object

AddRow(addRowObj)

DATASET에 데이터 추가

addRowObj
JSON 형태의 Object

RowData(rowIdx)

DATASET에서 rowIdx을 이용해 Row 데이터 반환

ReturnData
JSON 형태의 Object
rowIdx
데이터 인덱스 번호

RowData(rowIdx)

DATASET에서 rowIdx을 이용해 Row 데이터 반환

ReturnData
JSON 형태의 Object
rowIdx
데이터 인덱스 번호
