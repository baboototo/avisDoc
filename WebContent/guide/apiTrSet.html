TRSET API

Gauce TRSET Object를 기반으로 커스터마이징한 JavaScript Object이다.

* TRSET 사용시 유의사항
TRSET 생성시 유일한 ID값을 설정해야 한다.
TRSET Function 또는 속성값 사용시 대소문자를 정확하게 입력해야 한다

TRSET(trSetId)

new TRSET("trSetId") 생성시 trSetId 값을 필수로 설정해야 한다.

TRSET 속성 변수값

id
TRSET ID
Async
비동기 통신 여부(boolean)
Action
UI_HELPER_BASE 및 UI_HELPER_COMM 등...
KeyValue
DATASET KeyValue(JSP(I:DS01=condSet,O:DS02=dataSet01))
Parameters
EJB명 및 메소드명(ejbName=CUST0200,methodName=getCustomer)
IsShowBlock
데이터 조회시 메세지 "응답 대기중입니다..." 설정 여부(boolean) 기본값 true
ErrorCode
에러코드
ErrorMsg
에러메세지
TimeOut
데이터 통신 트랜젝션 시간설정(1000 = 1초) 기본값 120초

Post(isCallFun)

TRSET 설정 속성값을 이용해 데이터 이벤트를 처리

isCallFun
비동기 데이터 통신 설정시 데이터 이벤트가 완료 되면 호출되는 Function
